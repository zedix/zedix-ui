# Carousel

`zx-carousel` is based on the [Embla Carousel](https://www.embla-carousel.com/) library.

## Example 1

<Source
  dark
  code={`<style>

    --slide-gap: 1rem;

  <zx-carousel-item>Slide 1</zx-carousel-item>
  <zx-carousel-item>Slide 2</zx-carousel-item>
