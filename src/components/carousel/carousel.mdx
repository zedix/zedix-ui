# Carousel

`zx-carousel` is based on the [Embla Carousel](https://www.embla-carousel.com/) library, a [lightweight altenative to Swiper](https://npmtrends.com/@splidejs/splide-vs-embla-carousel-vs-slick-carousel-vs-swiper) with fluid motion and great swipe precision.

## Example 1

<Story of={Stories.Showcase} />

<Source
  dark
  code={`<style>
  zx-carousel {
    --slide-size: calc(100% / 3);
    --slide-gap: 1rem;
  }
<style>

<zx-carousel>
  <zx-carousel-item>Slide 1</zx-carousel-item>
  <zx-carousel-item>Slide 2</zx-carousel-item>

</zx-carousel>
`} />
